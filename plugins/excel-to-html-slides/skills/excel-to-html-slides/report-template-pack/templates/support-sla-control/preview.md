# Support SLA Control

Use for customer service and ticket tables where aging, response, resolution, and issue type drive decisions.

## Best Fit

- support tickets
- service work orders
- complaint handling exports
- case management tables

## Report Modules

1. Ticket volume, resolution rate, average response/handling time
2. SLA breach and aging indicators
3. Issue type and channel mix
4. Owner workload
5. Root-cause ranking
6. Staffing, escalation, and knowledge-base actions

## Visual Grammar

Command-center style with SLA badges, aging bars, issue cards, and red/amber exception emphasis.

## Recommendation Style

Operational and customer-protective: escalate old cases, add staffing, fix recurring product/service issue, update standard response.
