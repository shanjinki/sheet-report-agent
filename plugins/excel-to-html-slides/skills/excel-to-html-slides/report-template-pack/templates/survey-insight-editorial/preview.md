# Survey Insight Editorial

Use for survey or offline collection data where the report should feel like a synthesized insight brief.

## Best Fit

- questionnaire exports
- offline collection forms
- research feedback
- satisfaction surveys

## Report Modules

1. Response count and segment coverage
2. Rating or choice distribution
3. Segment comparison
4. Key finding cards
5. Representative comments when privacy-safe
6. Recommendation memo

## Visual Grammar

Editorial-brief: paper-like background, strong finding headlines, quote blocks, calm charts, readable paragraphs.

## Recommendation Style

Insight-led: "what this means", "what to change", "which segment to prioritize", "what to ask next".
