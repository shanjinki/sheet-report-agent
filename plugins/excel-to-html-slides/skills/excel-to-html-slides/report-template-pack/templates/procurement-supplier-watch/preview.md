# Procurement Supplier Watch

Use for supplier and purchase tables where concentration, delivery, and cost risks matter.

## Best Fit

- purchase order exports
- supplier scorecards
- delivery status sheets
- procurement cost trackers

## Report Modules

1. Purchase amount/order count/supplier count
2. Supplier concentration
3. Delivery or overdue status
4. Category/material spend mix
5. High-risk supplier list
6. Procurement follow-up actions

## Visual Grammar

Ops-ledger with supplier cards, concentration bars, overdue badges, ledger-like exception table.

## Recommendation Style

Procurement-focused: follow up supplier, diversify supply, renegotiate, expedite, validate receiving data.
