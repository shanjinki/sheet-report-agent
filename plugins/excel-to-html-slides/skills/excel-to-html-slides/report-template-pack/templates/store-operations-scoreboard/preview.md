# Store Operations Scoreboard

Use for store, branch, or regional operation tables.

## Best Fit

- store sales comparisons
- branch operations scorecards
- regional service tables
- store review/performance exports

## Report Modules

1. Store/region count and main KPI
2. Ranking and bottom performers
3. Region/category/service mix
4. Exception store list
5. Coaching or support priorities
6. Operational action plan

## Visual Grammar

Retail-pulse with scoreboard cards, ranking bars, region chips, issue badges.

## Recommendation Style

Field-operational: coach, support, investigate, replicate best practice, fix service or stock issue.
