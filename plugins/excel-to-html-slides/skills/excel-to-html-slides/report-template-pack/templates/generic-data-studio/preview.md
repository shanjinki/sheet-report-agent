# Generic Data Studio

Use when the table schema is unfamiliar or the user asks for broad exploratory analysis.

## Best Fit

- mixed business tables
- unknown CSV/XLSX exports
- first-pass data profiling
- messy internal trackers

## Report Modules

1. Dataset overview
2. Detected field roles
3. Missingness and duplicate-like risks
4. Top categorical distributions
5. Numeric summaries and outliers
6. Suggested next analyses

## Visual Grammar

Data-studio: neutral analytical surface, field cards, distribution panels, transparent assumptions.

## Recommendation Style

Methodical: explain what can be concluded now, what cannot, and what extra fields would make the report stronger.
