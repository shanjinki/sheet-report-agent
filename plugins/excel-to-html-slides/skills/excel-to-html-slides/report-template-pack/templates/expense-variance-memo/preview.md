# Expense Variance Memo

Use for finance tables where the audience needs budget, actual, variance, approval, or payment clarity.

## Best Fit

- expense reports
- budget trackers
- payment status sheets
- invoice exports

## Report Modules

1. Total amount, paid/unpaid, budget variance
2. Department/category/project mix
3. Monthly trend
4. Top variance items
5. Overdue or unapproved exceptions
6. Reconciliation and approval actions

## Visual Grammar

Boardroom-light, clean tables, variance bars, compact exception cards, restrained status colors.

## Recommendation Style

Precise: approve, reconcile, investigate variance, reallocate budget, verify source field before reporting.
