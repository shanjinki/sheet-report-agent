# Project Delivery Map

Use for portfolio, milestone, task, or project tracker tables.

## Best Fit

- PMO sheets
- project status trackers
- milestone exports
- task lists

## Report Modules

1. Project/task count and completion health
2. Status or stage distribution
3. Milestone/due-date risk
4. Owner/team workload
5. Blocked or delayed item list
6. Next decision/action map

## Visual Grammar

Command-center with roadmap-like sections, stage bars, owner cards, decision chips.

## Recommendation Style

Delivery-focused: unblock, reassign, revise deadline, split scope, escalate dependency.
