# Compliance Exception Register

Use when the report should prioritize abnormal, missing, overdue, or non-compliant records.

## Best Fit

- audit tables
- approval trackers
- data-quality registers
- finance/operations exception sheets

## Report Modules

1. Total records and exception count
2. Exception type distribution
3. Aging or overdue analysis
4. Owner/department accountability
5. Critical exception list
6. Remediation plan

## Visual Grammar

Boardroom-light with strict tables, red/amber severity badges, audit-friendly notes, clean source assumptions.

## Recommendation Style

Controlled: assign owner, set deadline, correct data, close overdue approval, document exception reason.
