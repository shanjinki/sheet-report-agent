# Review Sentiment Board

Use for rating/comment/feedback data where qualitative examples and quantitative distributions both matter.

## Best Fit

- ecommerce reviews
- store reviews
- customer satisfaction exports
- service feedback forms

## Report Modules

1. Review count, average rating, negative share
2. Rating distribution
3. Product/store issue ranking
4. Negative-topic themes
5. Representative excerpts when privacy-safe
6. Product/service improvement actions

## Visual Grammar

Friendly retail-pulse base with rating bars, sentiment chips, issue cards, excerpt blocks, and clear negative/positive contrast.

## Recommendation Style

Root-cause oriented: fix product issue, improve delivery/service process, reply to high-risk reviews, monitor recurring complaint themes.
