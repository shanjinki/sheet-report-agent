# Product Performance Matrix

Use for product/SKU/item tables where multiple metrics need tradeoff analysis.

## Best Fit

- SKU order exports
- product review tables
- product inventory sheets
- item-level performance trackers

## Report Modules

1. Product count and main metric
2. Top product ranking
3. Revenue/volume/refund/rating/inventory tradeoff
4. High opportunity and high risk products
5. Long-tail or concentration analysis
6. Product actions

## Visual Grammar

Retail-pulse with matrix cards, product ranking bars, opportunity/risk quadrants.

## Recommendation Style

Product-operational: promote, fix, delist, replenish, investigate refund/rating issue, bundle.
